# Gömülü Sistemler

Bu depo, bilgisayar mühendisliği eğitimim kapsamında aldığım Gömülü Sistemler dersi için Gowin EDA kullanarak geliştirdiğim FPGA projelerini, laboratuvar çalışmalarını ve donanım notlarımı içermektedir.

## 🖥️ Kullanılan Donanım
Bu depodaki tüm projeler aşağıdaki FPGA geliştirme kartı üzerinde test edilmiştir:

* **Kart Modeli:** Sipeed Tang Nano 9K 
* **FPGA Çipi:** GW1NR-LV9QN88PC6/I5

## 🛠️ Kullanılan Yazılım ve Geliştirme Ortamı
* **Geliştirme Ortamı (IDE):** Gowin EDA V1.9.12.01
* **Donanım Tanımlama Dili (HDL):** Verilog
* **Sentezleme ve Yükleme aracı:** Gowin Synthesis & Gowin Programmer
